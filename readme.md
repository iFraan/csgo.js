## Instalattion
### Dependencies
``
axios
``

To install use:
```shell
npm i csgo.js
```

> You'll need a **Steam API Key**, you can get one free [here](https://steamcommunity.com/dev/apikey)

> You must call **API.fetchUser** before using any other method.
